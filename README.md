# Overview
Implement conv2d layer in CNN step by step, regardless of efficiency.

Including:
- conv2d_with_stride
- conv2d_with_stride_padding
- conv2d_with_stride_padding_dilation
- conv2d_with_stride_padding_dilation_groups

## Dependancy
- NumPy
- PyTorch

## Usage:
```
mkdir -p build
cmake ..
make
cd ..
python test_conv2d.py
```
